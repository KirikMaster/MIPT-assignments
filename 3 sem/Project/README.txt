Structure
==================
Project.py - main, it is interface, supports 2 modes: webcam and preload video

webcam_part.py - ideologically main, video processing and symbols detection
by neuronet, visualising

Filters.py - functions that act on imput images

Neuronet.py - creating and training the neuronet

Predicting.py - using created in Neuronet.py neuronet predict the symbol in
a given image

Testfield.py - a draft for experiments and better understanding of CV
